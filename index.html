CRM

Tools

Developer tasks such as importing leads, social network login access and search engine keywords can be found under the tools section.

Importing Leads

Upload the .CSV file you want to import.

You can opt in to generae saved searches if required.

Update the status of the leads as required.

Assign the list of imported leads to an agent and/or group if applicable.

Click the "Import Leads" button.

Exporting Leads

Select the specific agent whose leads you want to export.

Add the data that you need to export for the leads to the list.

Update the .CSV file name if required.

Click the "Download" button to get a copy of the CSV file.

Listings

Adding a Custom IDX Search

Navigate to the add search page.

Create a title for the new custom search.

Add the required search panels and update configurations if needed.

Click the "Save" button, the new custom search should appear on the "IDX Searches" list.

Adding a Featured Listing

Search for a listing using the Address or MLS #.

Select the appropriate listing.

Click the "Save" button.

Client Upgrades

  • A new node is created in rewHUB.
    • The 4.8 server recipe must be used.
    • All new nodes should be made in GCE.
  • Programmer copies existing client's files and data to node created in previous step.
    • The upgrade.php tool is used to help: https://git.rewhosting.com/rew/upgrade
    • Git is used to perform a merge of the latest framework changes.
  • Site live team prepares to make the site live (DNS changes).
    • The upgrade.php tool is used for this: https://git.rewhosting.com/rew/_upgrade
  • Tutorial for upgrade.php tool: http://xerxes/wiki/REW_Upgrade_Tools

Q. How would you set up text messages for 4.8? do you have to contact support to add your number so that you can send out text messages?
A. Yes, you need to configure each beta site to work with the "REW Twilio" API. The tool to do this is here: http://twilio.rewhosting.com/

Instructions:

  • 1. Obtain REW API key by logging into the REW backend and navigate to: Leads > Tools > API Settings
  • 2. Go here and "Add New Account" using API key from previous step 1: http://twilio.rewhosting.com/
  • 3. Update the modules.yml to use the REW Twilio API key from step 2: REW_PARTNERS_TWILIO: abc123xyz
    • https://git.rewhosting.com/enterprise/deploy/blob/dev/config/environments/4.8-bcse/modules.yml#L43
